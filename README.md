# Internify: AI Internship Recommender

Internify is an AI-powered web application that helps students and aspiring professionals find the perfect internship. It recommends opportunities based on a user's field of study, skills, and interests by analyzing their profile and resume using the Google Gemini API.

## Features

- **AI-Powered Recommendations**: Leverages the Gemini API to analyze user profiles and resumes, providing a personalized "match score" for each internship.
- **Dynamic Internship List**: The list of available internships is dynamically generated by AI, ensuring a fresh and diverse set of opportunities.
- **Save for Later**: Users can save internships they are interested in for easy access later.
- **User Authentication**: Secure sign-up and login functionality powered by Supabase.
- **Resume Analysis**: Directly parses uploaded resumes (.pdf, .doc, .docx) to extract skills and experience for better matching.
- **Responsive Design**: A clean, modern UI that works seamlessly on desktop and mobile devices.
- **Dark Mode**: A sleek dark theme for comfortable viewing in low-light environments.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **AI**: Google Gemini API (`gemini-2.5-flash`)
- **Backend**: Netlify Functions (for proxying API calls to Gemini)
- **Authentication & DB**: Supabase
- **Deployment**: Netlify

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [Netlify CLI](https://docs.netlify.com/cli/get-started/): To run the serverless functions locally. You can install it after cloning the repo by running `npm install`.

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/CharanRakindi/internify.git
    cd internify
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up environment variables:**

    Copy the example environment file:
    ```bash
    cp .env.example .env
    ```

    Now, open the `.env` file and add your credentials:

    -   `GEMINI_API_KEY`: Your Google Gemini API key. You can get one from [Google AI Studio](https://aistudio.google.com/app/apikey).
    -   `SUPABASE_URL`: Your Supabase project URL.
    -   `SUPABASE_ANON_KEY`: Your Supabase project's `anon` (public) key.

    Your `.env` file should look like this:
    ```
    GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
    SUPABASE_URL="https://your-project-id.supabase.co"
    SUPABASE_ANON_KEY="your-public-anon-key"
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```
    This command starts the Netlify development server, which will serve your static files and run the serverless function locally. The application should now be running at `http://localhost:8888`.

## How It Works

The application consists of two main parts:

1.  **React Frontend**: The user interface built with React and styled with Tailwind CSS. It handles user input, authentication, and displaying internship data.
2.  **Netlify Function (`gemini-proxy.ts`)**: A serverless function that acts as a secure backend proxy. It takes requests from the frontend, adds the secret `API_KEY`, and communicates with the Google Gemini API. This is done to avoid exposing the API key in the browser.

The workflow is as follows:
- The app first calls the `generate` action on the backend to get a fresh list of internships.
- The user fills out the form and uploads their resume.
- The frontend sends the user's profile and the full list of internships to the `recommend` action on the backend.
- The Netlify function analyzes the user's resume and profile against all internships using the Gemini API.
- The function returns a sorted list of the best-matching internships with a `matchScore` and `reasoning`, which is then displayed to the user.

## Deployment

This project is configured for easy deployment on [Netlify](https://www.netlify.com/).

### Steps to Deploy:

1.  **Prepare your repository:**
    - Ensure `.env` is in `.gitignore` (it already is)
    - Commit all code to GitHub
    ```bash
    git add .
    git commit -m "Ready for production deployment"
    git push origin main
    ```

2.  **Connect to Netlify:**
    - Go to [Netlify](https://app.netlify.com)
    - Click "Add new site" → "Import an existing project"
    - Select your GitHub repository

3.  **Configure environment variables:**
    - In Netlify Dashboard, go to `Site settings` → `Build & deploy` → `Environment`
    - Add these variables:
      - `GEMINI_API_KEY`: Your Google Gemini API key
      - `SUPABASE_URL`: Your Supabase URL
      - `SUPABASE_ANON_KEY`: Your Supabase anonymous key

4.  **Set build settings:**
    - Build command: `npm run build`
    - Publish directory: `dist`
    - Functions directory: `netlify/functions`

5.  **Deploy:**
    - Click "Deploy" and Netlify will automatically build and deploy your application
    - Your site will be live at `https://your-site-name.netlify.app`

### Production Checklist:

- ✅ All environment variables are set in Netlify Dashboard

- ✅ API keys are not exposed in the repository
- ✅ `npm run build` completes successfully
- ✅ CORS is properly configured if needed
